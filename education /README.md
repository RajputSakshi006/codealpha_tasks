# 📚 Education Analytics Dashboard (Power BI)

## 📌 Project Overview
This Power BI dashboard analyzes student performance and resource allocation to support data-driven educational decision-making. It provides insights into academic performance, attendance, resource utilization, and institutional trends.

---

## 🎯 Objectives
- Analyze student academic performance.
- Monitor attendance trends.
- Compare department-wise results.
- Evaluate resource allocation.
- Identify high-performing and low-performing departments.
- Support strategic educational planning.

---

## 📊 Dashboard Features
- Total Students
- Average Marks
- Average Attendance
- Pass Percentage
- Department-wise Performance
- Subject-wise Marks Analysis
- Resource Allocation
- Student Performance Trend
- Attendance Analysis
- Interactive Filters (Department, Semester, Gender)

---

- DAX
